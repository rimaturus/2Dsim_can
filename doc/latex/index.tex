\hypertarget{index_intro_sec}{}\doxysection{Introduction}\label{index_intro_sec}
This documentation describes a small car simulation that uses\+:
\begin{DoxyItemize}
\item \href{https://liballeg.org/}{\texttt{ Allegro}} for graphics
\item \href{https://github.com/yaml/libyaml}{\texttt{ Lib\+YAML}} for parsing configuration files
\item \href{https://man7.org/linux/man-pages/man7/pthreads.7.html}{\texttt{ pthread}} for concurrency
\item A custom periodic tasks library ({\ttfamily ptask})
\end{DoxyItemize}\hypertarget{index_build_sec}{}\doxysection{Build}\label{index_build_sec}
Typical compilation might look like\+: 
\begin{DoxyCode}{0}
\DoxyCodeLine{gcc -\/g -\/O3 -\/o 2Dsim\_can 2Dsim\_can.c ptask/ptask.c `allegro-\/config -\/-\/libs` -\/lyaml -\/lm -\/lpthread}

\end{DoxyCode}
\hypertarget{index_usage_sec}{}\doxysection{Usage}\label{index_usage_sec}
Run the resulting executable using {\ttfamily sudo} to enable RT clock. Use arrow keys to control the car, and ESC to quit.

Detailed Description

Implements an enhanced GraphSLAM algorithm.

The GraphSLAM class maintains a graph-based representation of the robot's poses and landmarks in the environment. It provides methods to add poses, landmarks, and measurements, and performs optimization to compute the posterior probability of the robot's trajectory and landmark positions.

\doxysubsection{Detailed Description}
Global constants, variables, and structures for the simulation and visualization system. 

This header file declares all global macros, extern variables, and data types required across multiple modules for task management, visualization, and perception in a 2D simulation environment.\hypertarget{globals_8h_tasks_constants}{}\doxysubsection{Tasks Constants}\label{globals_8h_tasks_constants}

\begin{DoxyItemize}
\item Defines task periods (in milliseconds)\+:
\begin{DoxyItemize}
\item PERCEPTION\+\_\+\+PERIOD\+: Period for the perception task.
\item TRAJECTORY\+\_\+\+PERIOD\+: Period for the trajectory planning task.
\item CONTROL\+\_\+\+PERIOD\+: Period for the control task.
\item DISPLAY\+\_\+\+PERIOD\+: Period for the display update task.
\end{DoxyItemize}
\item Defines deadlines (in milliseconds), set equal to the corresponding task periods\+:
\begin{DoxyItemize}
\item PERCEPTION\+\_\+\+DEADLINE, TRAJECTORY\+\_\+\+DEADLINE, CONTROL\+\_\+\+DEADLINE, DISPLAY\+\_\+\+DEADLINE.
\end{DoxyItemize}
\item Defines task priorities (lower number indicates higher priority)\+:
\begin{DoxyItemize}
\item PERCEPTION\+\_\+\+PRIORITY, TRAJECTORY\+\_\+\+PRIORITY, CONTROL\+\_\+\+PRIORITY, DISPLAY\+\_\+\+PRIORITY.
\end{DoxyItemize}

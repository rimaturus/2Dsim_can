\doxysubsection{Detailed Description}
Declaration of task thread functions for the simulation. 

This header file declares thread functions for various tasks in the simulation\+:
\begin{DoxyItemize}
\item Perception\+: Processes sensor or simulation data to interpret the environment.
\item Trajectory\+: Computes and updates planned paths or trajectories.
\item Control\+: Implements control logic that may handle both manual and autonomous inputs.
\item Display\+: Manages rendering of the simulation\textquotesingle{}s visual output.
\end{DoxyItemize}

Each task is designed to be executed in its own thread, and the functions follow the prototype required by POSIX thread routines.

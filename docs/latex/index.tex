 \hypertarget{index_autotoc_md1}{}\doxysection{Project Overview}\label{index_autotoc_md1}
This project is a 2D simulation of a Formula SAE (FSAE) vehicle, designed to model vehicle dynamics, perception, trajectory planning, and control systems. It provides a platform for testing and visualizing autonomous driving algorithms in a simplified environment.\hypertarget{index_autotoc_md2}{}\doxysection{Key Features}\label{index_autotoc_md2}

\begin{DoxyItemize}
\item {\bfseries{Vehicle Dynamics\+:}} Simulates basic vehicle movement based on pedal and steering inputs.
\item {\bfseries{Li\+DAR-\/based Perception\+:}} Implements cone detection using simulated Li\+DAR measurements and a circle Hough transform.
\item {\bfseries{Trajectory Planning\+:}} Generates a driving trajectory by connecting detected cones.
\item {\bfseries{Autonomous Control\+:}} Includes an autonomous control system to follow the planned trajectory.
\item {\bfseries{Manual Control\+:}} Allows keyboard control of the vehicle.
\item {\bfseries{Visualization\+:}} Uses the Allegro library to provide a 2D graphical representation of the simulation, including the car, track, cones, and planned trajectory.
\item {\bfseries{Periodic Tasks\+:}} Utilizes a periodic task scheduler for managing different simulation components.
\end{DoxyItemize}\hypertarget{index_autotoc_md3}{}\doxysection{Project Structure}\label{index_autotoc_md3}
The project is organized into several modules\+:

\item $\ast$$\ast${\ttfamily \mbox{\hyperlink{trajectory_8c}{src/trajectory.\+c}}}\+:$\ast$$\ast$ Plans the driving trajectory based on detected cones.
\item $\ast$$\ast${\ttfamily \mbox{\hyperlink{control_8c}{src/control.\+c}}}\+:$\ast$$\ast$ Implements control algorithms for both manual and autonomous driving.
\item $\ast$$\ast${\ttfamily \mbox{\hyperlink{display_8c}{src/display.\+c}}}\+:$\ast$$\ast$ Manages the graphical display of the simulation.
\item $\ast$$\ast${\ttfamily \mbox{\hyperlink{tasks_8c}{src/tasks.\+c}}}\+:$\ast$$\ast$ Defines the periodic tasks for perception, trajectory planning, control, and display.

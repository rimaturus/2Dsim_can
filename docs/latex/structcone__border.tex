\doxysubsection{Detailed Description}
Represents a border of a cone as seen in a Li\+DAR scan. 

This structure collects indices from the Li\+DAR scan that correspond to the boundaries of a cone. It also stores the color information associated with that cone border.

Members\+:
\begin{DoxyItemize}
\item angles\+: An array of indices (angles) in the Li\+DAR scan corresponding to the same cone border.
\item color\+: Color of the cone border. 
\end{DoxyItemize}
